ทำข้อสอบข้อ 2

1. Model
   1.1.) model.js - เก็บ Data ตรวจสอบความถูกต้องและ Logic ของการจัดการสินค้าและ Track ยอด

2. View
   2.1.) view.js - อัพเดท UI โดยการแสดงข้อความ, List ของสินค้า และยอด
   2.2.) style.css - เพิ่ม Styling ให้หน้า UI ดูมีสีสัน เป็นมิตรมากขึ้น
   2.3.) index.html - โครงสร้างของ UI ได้แก่ Form กับ Container ในการแสดงข้อมูล

3. Controller
   3.1.) controller.js - เช็ค Interaction ของ User, Input ของ Process, ผสาน Model กับ View 
   3.2.) app.js - ใช้เริ่มต้น App โดยการสร้าง Event listener และแสดง Initial data
   3.3.) server.js - Backend server สำหรับ Application
